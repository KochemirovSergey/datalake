# ЧТЗ Этап 2: Слой Нормализации (Normalization)

## 1. Задача
Создание слоя `silver_raw` (Узлы `n_*`). Очистка сырых данных от мусора, нормализация текстов и интеграция всех исторических багфиксов.

## 2. Исходное состояние
- В `transformations/bronze_normalized/` лежат скрипты (`region_normalizer.py`, `year_normalizer.py`), которые читают конфиги с диска.
- Исторические фиксы (из `BUGFIXES_AUDIT.md`) разбросаны по всем слоям.

## 3. Целевое состояние
- Все узлы `n_*` зависят от выходов `1_*` и возвращают очищенные `pd.DataFrame`.
- Справочники (регионы, уровни) приходят как аргументы (Dependency Injection).
- Никаких regex или ручных фиксов (вроде "если Хакасия, то...") в последующих слоях быть не должно.

## 4. Обязательные шаги очистки (Пайплайн)

Каждый узел `n_*` должен последовательно применять следующие функции:

### Шаг 4.1. Row Gate (Фильтрация мусора)
Из сырых данных **ДОЛЖНЫ БЫТЬ УДАЛЕНЫ** все строки, являющиеся агрегатами:
- Регионы с названиями "Российская Федерация", "Всего по РФ".
- Строки, где код региона `RU-FED` или тип региона `Федеральный округ`.
- (Источник бага: двойной счет сумм в витринах).

### Шаг 4.2. Текстовая санитария (OCR Fixes)
Применить функцию ко всем строковым полям перед маппингом:
- Замена латинских 'c', 'o', 'a', 'p', 'x', 'e' на кириллические эквиваленты.
- Приведение к нижнему регистру и `strip()`.
- (Источник бага: ошибки маппинга из-за опечаток в Excel).

### Шаг 4.3. Маппинг регионов и годов
- `LEFT JOIN` очищенных названий с таблицей `regions` (переданной из `1_extract`).
- Использовать колонку `aliases` (синонимы) из справочника для сложных случаев (например, "Херсонская обалсть" -> "Херсонская область").
- **Важно:** Строки, где `region_id` остался `NULL` после маппинга, не удалять молча, а логировать (через `context.log.warning` Дагстера).

### Шаг 4.4. Парсинг возрастов (Узел `n_age_dict`)
Создать отдельную трансформацию, которая превращает строки вида "от 2 до 3 лет" в три колонки: `age_min`, `age_max`, `age_category`. (Источник бага: разбросанные regex по файлам).
